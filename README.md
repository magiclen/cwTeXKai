cwTeXKai
=================================

## Usage

Just add woff2 font files into your CSS.

e.g.

```css
p {
    font-family: local('cwTeXKai'), local('cwTeXKai Medium'), url('/path/to/cwTeXKai-zhonly.woff2') format('woff2');
    font-size: 1.16rem;
}
```
